# Alfred

An Elixir library for integrating with [Alfred](http://www.alfredapp.com).

## Installation

If [available in Hex](https://hex.pm/docs/publish), the package can be installed
by adding `alfred` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:alfred, "~> 0.1.0"}]
end
```
